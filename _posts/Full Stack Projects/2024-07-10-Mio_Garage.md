# 🚗 Mio Garage - Switzerland

**Mio Garage** is a sophisticated parking unit management and reservation system tailored for the Swiss market. It provides users with a seamless experience for booking parking spaces, offering flexible options for both short-term (hourly) and long-term (daily) stays.

---

## Features

- **Flexible Booking Options:** Users can reserve parking units based on their specific needs, choosing between hourly or daily rates.
- **Advanced Admin Dashboard:** A robust control panel that allows administrators to manage unit data, update pricing, and upload high-quality images.
- **Real-Time Availability:** Integrated with Firebase to provide live updates on unit status and availability.
- **Dynamic Content Management:** Easily modify unit descriptions, locations, and specifications through the admin interface.
- **Secure Authentication:** Role-based access control to ensure secure management of bookings and user data.
- **Responsive & Modern Design:** A clean, user-friendly interface optimized for all devices, including mobile and desktop.

---

## Technologies Used

- **HTML5 & CSS3:** For a modern, responsive, and visually appealing user interface.
- **JavaScript (ES6+):** For handling dynamic interactions and real-time UI updates.
- **Firebase Authentication:** For secure user and admin login management.
- **Firebase Firestore:** A cloud-hosted NoSQL database for real-time data synchronization.
- **Firebase Storage:** For managing and hosting high-resolution images of the parking units.
